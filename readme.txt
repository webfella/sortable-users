== Description ==

Easily sort users by dragging and dropping them within the user admin section with jQuery UI

== Installation ==

1. Upload the `sortable-users` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Viola!

== Frequently Asked Questions ==

= How can I leverage the order from within a template? =

The order data is stored under the custom meta key `user_order`
